# En mi cocina - Aplicación React para la Planificación de Comidas

## Descripción

Esta aplicación de React permite a los usuarios planificar su menú semanal y generar una lista de compras. Ofrece tres componentes principales:

- **Menu**: Donde puedes añadir las comidas y cenas para cada día de la semana.
- **Recipes**: Este componente no está incluido en el código proporcionado, pero podrías añadir recetas o enlaces a ellas aquí.
- **ShoppingList**: Aquí puedes añadir los artículos que necesitas comprar.

Todos los datos se guardan automáticamente en el LocalStorage, así que no tienes que preocuparte por perder tu información.

Este proyecto forma parte de mi práctica para refrescar y aprender más sobre desarrollo web y React.

## Cómo empezar

### Prerrequisitos

- Node.js
- npm

### Instalación

1. Clona el repositorio.

2. Navega hasta el directorio del proyecto.

3. Instala las dependencias.
npm install

4. Ejecuta la aplicación.
npm start


## Uso

### Menú Semanal

1. Para añadir una comida o cena, haz clic en el cuadro de texto correspondiente e introduce el nombre del platillo.
2. Para eliminar una comida o cena, marca la casilla de verificación que aparece junto al nombre del platillo.

### Lista de la Compra

1. Añade elementos a la lista introduciendo el nombre en el cuadro de texto y haciendo clic en "Agregar".
2. Para eliminar elementos, marca la casilla de verificación que aparece junto al nombre del elemento.
